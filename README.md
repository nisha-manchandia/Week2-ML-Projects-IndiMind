# Week2-ML-Projects-IndiMind

## Project Title

Week 2 ML Projects — Linear Regression & Classification

## Objective

Build simple machine learning examples for linear regression and binary classification, and document the project workflow.

## Linear Regression

### Dataset

- Study Hours: [1, 2, 3, 4, 5]
- Marks: [35, 45, 50, 60, 70]
- The dataset is small and used for a basic regression demonstration.

### File

- `linear regression/linear regression.py`

### How to Run

1. Open the project folder in your Python environment.
2. Install NumPy if it is not already installed:
   ```powershell
   pip install numpy
   ```
3. Run the linear regression script:
   ```powershell
   python "linear regression\linear regression.py"
   ```
4. Review the output for mean values, slope, intercept, predicted marks, and mean squared error.

## Classification

### Dataset

- Study Hours: [1, 2, 3, 4, 5]
- Labels: [0, 0, 0, 1, 1] (0 = fail, 1 = pass)
- The dataset is a small binary classification example.

### File

- `classification/classification.py`

### How to Run

1. Open the project folder in your Python environment.
2. Install NumPy if it is not already installed:
   ```powershell
   pip install numpy
   ```
3. Run the classification script:
   ```powershell
   python classification\classification.py
   ```
4. Review the output for model parameters, raw predictions, predicted classes, and accuracy.
